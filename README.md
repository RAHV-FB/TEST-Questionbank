# RAHV Question Bank

A static GitHub Pages site for viewing, editing, and curating science exam questions with mark schemes.

## Sites

| Page | URL | Purpose |
|------|-----|---------|
| **Home** | `/` | Landing page with navigation |
| **Tester** | `/tester/` | Browse and edit questions |
| **Checker** | `/checker/` | Review changes side-by-side |

## Features

- 📋 **Question types**: MCQ, Short Answer, Long Answer, Structured
- 🖼️ **Images**: Inline diagrams served from `/images/`
- ✅ **Mark schemes**: Collapsible per question/part
- ✎ **Edit mode**: Modify type, text, marks, mark scheme — no CSV changes
- 🔗 **Grouping**: Combine questions into multi-part sets with custom ordering
- ⚠ **Flagging**: Mark questions as needing more information
- 📊 **Checker**: Side-by-side diff of original vs. edited
- 💾 **GitHub sync**: All changes saved to `data/edits/` via GitHub API

## CSV Format

| Column | Description |
|--------|-------------|
| `subject` | Subject name (Biology, Chemistry, Physics) |
| `topic` | Main topic |
| `subtopic` | Subtopic |
| `question_index` | Integer (groups multi-part questions together) |
| `sub_index` | Part letter (a, b, c…) — empty for standalone |
| `question_type` | MCQ / Short Answer / Long Answer / Structured |
| `question_text` | HTML allowed: `<b>`, `<sub>`, `<sup>`, `<br>`, etc. |
| `option_a/b/c/d` | MCQ options |
| `correct_option` | A, B, C, or D |
| `markscheme` | Mark scheme text (HTML allowed) |
| `marks` | Number of marks |
| `image_path` | Filename in `/images/` (e.g. `bio_cell_01.png`) |
| `difficulty` | Easy / Medium / Hard |
| `year` | Source exam year |
| `paper` | Paper reference (P1, P2, etc.) |
| `notes` | Internal notes |

## GitHub Sync Setup

1. Create a GitHub [Personal Access Token](https://github.com/settings/tokens) with **Contents: Read and Write**
2. On the Tester or Checker page, click **⚙ Settings**
3. Enter your token, owner (`rahv-questionbank`), repo, and branch
4. Click **Save Settings**

Changes are saved to `data/edits/` — original CSV files are **never modified**.

## Adding Images

Place image files in the `/images/` folder and reference them in the CSV `image_path` column as just the filename (e.g., `bio_cell_01.png`).

## Adding Subjects

1. Add a CSV file to `data/questions/`
2. Add an entry to `data/manifest.json`
